## Возможности

- **Автоматическое определение формата разделителей**
- **Проверка данных** — валидация ID, имён и значений
- **Многоформатный вывод** — вывод в TXT, Markdown, AsciiDoc форматах, а также отображение в консоль
- **Гибкая обработка чисел** — поддержка как точки, так и запятой как десятичного разделителя
- **Вывод в файл и консоль**

## Использование

### Базовая команда

```bash
python -m src -i <input-file>
```

### Параметры

| Параметр | Краткая форма | Описание | Обязательный | По умолчанию |
|----------|---------------|---------|--------------|--------------|
| `--input` | `-i` | Путь к входному файлу | Да | — |
| `--output` | `-o` | Путь к файлу отчёта | Нет | `result.txt` |
| `--console` | `-c` | Вывести отчёт в консоль | Нет | Отключено |

### Примеры

#### Обработать файл и сохранить отчёт
```bash
python -m src -i input.txt
# Результат сохранится в result.txt
```

#### Обработать файл, вывести отчёт в консоль и сохранить его
```bash
python -m src -i input.txt -c
# Результат сохранится в result.txt
```

#### Сохранить отчёт в TXT
```bash
python -m src -i input.txt -o output.txt
```

#### Сохранить отчёт в Markdown
```bash
python -m src -i input.txt -o output.md
```

#### Сохранить отчёт в AsciiDoc
```bash
python -m src -i input.txt -o output.adoc
```

## Формат входных данных

Входной файл должен содержать CSV данные с тремя полями:

```
<ID>,<Name>,<Value>
```

Где:
- **ID** — целое число (отрицательные числа допускаются)
- **Name** — строка (может содержать несколько слов)
- **Value** — число с плавающей точкой (поддерживает запятую и точку как разделитель)

### Примеры корректного ввода

```csv
1,Alice,100
2,Bob,200.5
3,Charlie Kirk,300
```


## Форматы вывода

- TXT (текстовый формат, по умолчанию)

- Markdown (`.md`)

- AsciiDoc (`.adoc`)
